# Сложно сосредоточиться

Информационный лендинг о причинах, которые мешают концентрироваться, и простых способах вернуть внимание к задаче. Проект сверстан по макету учебного курса и показывает адаптивную вёрстку, работу с темами и аккуратную семантическую структуру страницы.

## Репозиторий

(https://github.com/vlasenkoElena-mag/slozhno-sosredotochitsya-fd)

## Что есть в проекте

- адаптивная главная страница с шапкой, текстовыми секциями, галереей и футером;
- светлая, тёмная и автоматическая тема оформления;
- фиксированный фон с декоративными слоями;
- тематическое оформление кнопок и ссылок;
- CSS Grid и Flexbox для построения сетки;
- изображения с `loading="lazy"` для галереи.

## Стек

- HTML5;
- CSS3;
- JavaScript для переключения темы;

## Структура

- `index.html` — основная разметка страницы;
- `styles/variables.css` — CSS-переменные;
- `styles/style.css` — основные стили;
- `styles/light.css` и `styles/dark.css` — темы;
- `styles/media.css` — адаптивные правки;
- `styles/ui_kit.css` — интерактивные состояния;
- `scripts/script.js` — переключение темы.

## Как открыть

1. Откройте `index.html` в браузере.
